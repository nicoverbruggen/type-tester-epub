# Trevelyan's Type Tester

## What is this?

A story about Tanya, type designer with dyslexia who's far too perfectionistic for her own good.

The original concept and the various typographic examples were generated with the help of a large language model, but the copy has been edited and rewritten by me, Nico Verbruggen. 

I consider it a test document with some whimsy, not a fully qualified creative work.
